# AI进场助手

这是一个 Python + Streamlit 写的加密货币交易辅助工具。它会把多周期行情分析整理成普通人能直接看懂的每日交易计划：

- 今日交易一句话：一打开就知道今天怎么处理
- 市场状态：强势上涨 / 震荡等待 / 弱势下跌
- 方案A：激进交易（抄底）
- 方案B：稳健交易（趋势确认）
- 必须做这一单模式：支持“今天必须做”和“现在必须进”
- 强制模式会给出具体入场价、止损、目标、数量、仓位价值、保证金、止损亏损和目标盈利
- “现在必须进”会使用当前价、自动降低风险、限制杠杆不超过3x，并提示这不是最佳位置
- 欧易填写指南：逐仓、杠杆、方向、委托类型、价格、数量、仓位、保证金、止损和止盈
- 今天不要做什么
- 资金管理：最多亏损、建议最大仓位、推荐杠杆
- 合约下单助手：输入人民币本金、方向、入场价、止损价、杠杆和风险比例，直接输出交易所该填什么

它只做分析和风险计算，不连接交易所，不自动下单。

## 主要功能

- 输入 BTC/USDT、ETH/USDT、SOL/USDT 等币种
- 自动分析 15分钟、1小时、4小时、日线
- 自动识别支撑位、压力位、前高、前低、MA20、MA60
- 首页像交易员每日计划，不显示一堆指标
- 即使不适合立刻交易，也会给出等待区域和重新评估条件
- 输出等待区域、触发条件、建议入场价、止损、目标、盈亏比、风险等级
- 输入本金和单笔最大风险后，自动计算今日最多亏损、建议最大仓位、推荐杠杆
- 币种、本金、风险比例放在首页按钮同一个表单里，ETH/USDT、BTC/USDT、SOL/USDT 都可以直接分析
- 下单助手会输出约等于USDT本金、本单最多亏损、USDT仓位、币数量、保证金、止损价和两档目标价
- 下单助手新增“这单赚亏一眼看懂”：显示止损亏损、目标1/2盈利、人民币换算、手续费、扣费后实际盈亏
- 支持手动填写 USDT/CNY 汇率、目标价1/2、手续费率
- 如果杠杆太高、单笔风险太高、保证金不足或止损方向填反，会直接提示不要开
- 原行情图、RSI、MACD、均线、回测放在“高级指标”页面
